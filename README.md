# Entrega-2
Segunda entrega para el curso  "Computacion para Econocmistas" de la Universidad de Costa Rica

1. Numeros pares
El usuario ingresa dos numeros y el programa imprime todos los numeros pares entre ellos, inclusive.

2. Sumatoria
El usuario ingresa un numero n y el programa le devuelve la sumatoria de i=1 hasta n

3. Raices
El usuario ingresa tres numeros y el programa le devuelve la solucion a la ecuación de la forma ax² + bx + c = 0

4. Triangulo
El usuario ingresa el tamaño de cada lado y el programa imprime si forman, o no, un triangulo
